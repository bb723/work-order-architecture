How Each Channel Works

Tenants can reach out via text, phone call, or the web — each triggers a different AI-powered flow.

💬

SMS Intake

Tenants text their issue. Claude AI has a natural conversation to gather details, then creates a work order automatically.

  1. 1Tenant sends a text message
  2. 2Twilio delivers it to our backend
  3. 3Claude AI chats back to gather info — name, location, what's wrong
  4. 4AI assesses urgency (emergency = instant escalation)
  5. 5Ticket created and tenant gets confirmation
📞

Voice Intake

Tenants call and speak to an ElevenLabs AI voice agent that listens, asks questions, and files the ticket — no hold music needed.

  1. 1Tenant calls the property number
  2. 2Twilio connects to ElevenLabs voice AI
  3. 3AI agent has a real conversation with the tenant
  4. 4Agent files the ticket via backend tools
  5. 5Tenant gets spoken confirmation on the call
🧠

Smart Assignment

A manager chats with the AI agent to assign work orders. It considers employee skills, workload, availability, and urgency.

  1. 1Manager asks the AI to assign open tickets
  2. 2AI checks skills, workloads, and schedules
  3. 3Suggests best-fit employees with reasoning
  4. 4Manager can correct — AI learns from feedback
  5. 5Employee gets an SMS with assignment details
📸

Completion & Feedback

After the time block ends, employees get a follow-up text. They reply with photos, then tenants rate the work.

  1. 1Scheduler sends follow-up at noon or 5 PM
  2. 2Employee replies with photos via MMS
  3. 3Work marked complete, tenant surveyed
  4. 4AI analyzes feedback for sentiment & score
  5. 5Full audit trail stored for every step
🚨

Emergency Escalation

When the AI detects an emergency (burst pipe, no heat in Maine winter, gas leak), it immediately calls the property manager.

  1. 1AI detects emergency urgency
  2. 2Ticket created with emergency flag
  3. 3Twilio places outbound call to property manager
  4. 4Automated voice announces the emergency details
  5. 5Manager can respond immediately
🖥️

Management Dashboard

The web app gives managers a live view of all tickets, employees, schedules, and the AI chat interface — all in one place.

  1. 1Dashboard: live ticket stats, filters, search
  2. 2Employees: manage team & set skill levels
  3. 3Schedule: weekly/daily calendar of assignments
  4. 4Agent Chat: talk to AI for smart assignments
  5. 5Settings: manage properties & contacts
